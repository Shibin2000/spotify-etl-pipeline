# Spotify ETL Pipeline

A data engineering project that extracts, transforms, and loads Spotify track data.

## What it does
- Extracts 32,000+ real Spotify songs from dataset
- Transforms and cleans the data using Python and Pandas
- Loads into MySQL database
- Analyzes popularity trends with SQL
- Visualizes top 10 most popular artists

## Tools Used
- Python
- Pandas
- MySQL
- Matplotlib
- Google Colab

## Key Findings
- Loaded 32,828 songs successfully
- Dance Monkey by Tones and I scored 100 popularity
- 1,512 songs classified as Very Popular
